# Color System

## Development

### Environment

- **Runtime**: Node.js v24+ (Required for native TypeScript support)
- **Package Manager**: pnpm

### Running Scripts

This project uses Node.js 24's native TypeScript support. Do not use `ts-node`.
Run scripts directly with `node`:

```bash
node scripts/generate-tokens.ts
```

Or use the package scripts:

```bash
pnpm solve
```
