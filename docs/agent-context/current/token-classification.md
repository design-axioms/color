# Token Classification (Epoch 34: Phase 2)

This document classifies the CSS variables generated by the system into "Public API" and "Internal Plumbing".

## Public API (Semantic Tokens)

These tokens are intended for direct use by consumers (via CSS variables or utility classes). They represent the "Contract" of the design system.

### Surfaces

- `--axm-surface-token`: The background color of the current surface.

### Text

- `--axm-text-high-token`: High-emphasis text color (Ink).
- `--axm-text-subtle-token`: Medium-emphasis text color.
- `--axm-text-subtlest-token`: Low-emphasis text color.

### Borders

- `--axm-border-dec-token`: Decorative border color (low contrast).
- `--axm-border-int-token`: Interactive border color (medium contrast).

### Primitives

- `--axm-shadow-sm`, `--axm-shadow-md`, `--axm-shadow-lg`, `--axm-shadow-xl`: Elevation shadows.
- `--axm-focus-ring-color`: The color of the focus ring.
- `--axm-highlight-ring-color`: The color of the selection ring.
- `--axm-highlight-surface-color`: The background color of selected text/elements.

### Data Visualization

- `--axm-chart-1` through `--axm-chart-10`: Categorical colors for charts.

### Key Colors (Optional)

- `--axm-key-{name}-color`: The raw key colors defined in the config (e.g., `--axm-key-brand-color`).

## Internal Plumbing (Private)

These tokens are used for calculation, state management, and the "Reactive Pipeline". They should NOT be used directly by consumers, as their implementation details may change.

### State & Metadata

- `--_axm-inverted-surfaces`: A comma-separated list of inverted surface selectors (used by JS runtime).

### Key Color Components

- `--_axm-hue-{name}`: The hue component of a key color.
- `--_axm-chroma-{name}`: The chroma component of a key color.

### Surface Calculation (Local Scope)

- `--_axm-local-light-h`: The base hue for the surface in light mode.
- `--_axm-local-light-c`: The base chroma for the surface in light mode.
- `--_axm-local-dark-h`: The base hue for the surface in dark mode.
- `--_axm-local-dark-c`: The base chroma for the surface in dark mode.

### Surface Calculation (Resolved)

- `--_axm-surface-light-h`: The final resolved hue for the surface in light mode.
- `--_axm-surface-light-c`: The final resolved chroma for the surface in light mode.
- `--_axm-surface-dark-h`: The final resolved hue for the surface in dark mode.
- `--_axm-surface-dark-c`: The final resolved chroma for the surface in dark mode.

### Reactive Pipeline (Context)

- `--_axm-base-chroma`: The base chroma for the current context.
- `--_axm-base-hue`: The base hue for the current context.
- `--_axm-hue-brand`: The brand hue for the current context.
- `--_axm-chroma-brand`: The brand chroma for the current context.
- `--_axm-surface-transparency`: The transparency level for the surface.
- `--_axm-surface-chroma-adjust`: Adjustment factor for surface chroma.
- `--_axm-hue-adjust`: Adjustment factor for hue.

### Computed Values (Engine)

- `--_axm-computed-surface`: The final calculated background color.
- `--_axm-computed-fg-color`: The final calculated foreground color.
- `--_axm-computed-border-dec-color`: The final calculated decorative border color.
- `--_axm-computed-border-int-color`: The final calculated interactive border color.
- `--_axm-computed-surface-C`: Intermediate chroma value.
- `--_axm-computed-surface-H`: Intermediate hue value.
- `--_axm-computed-fg-C`: Intermediate foreground chroma.
- `--_axm-computed-fg-H`: Intermediate foreground hue.
- `--_axm-computed-border-alpha`: Intermediate border alpha.

### Indirection Sources

- `--_axm-text-lightness-source`: The source token for text lightness (e.g., set by `.text-subtle`).
- `--_axm-text-chroma-source`: The source token for text chroma.
- `--_axm-text-hue-source`: The source token for text hue.
