# Visual Description: Tokens Page (Desktop / Dark)

## 1. Token Tables

- **Inline Code**: Token names have a dark background.
  - **Contrast**: Good contrast against the table cell background.
- **Borders**: Table borders are visible but subtle.

## 2. Code Blocks

- **Section**: "Composition".
- **Issue**: The code block background is significantly lighter/bluer (`#24292e` approx) than the page background (black/very dark gray).
  - **Effect**: Creates a "cutout" or "sticker" effect that breaks the immersion of the dark theme.
  - **Consistency**: This confirms the finding from the Home Page and Concepts Page.

## 3. Sidebar

- **Active State**: "Tokens" link is highlighted in blue.
- **Issue**: Blue color on dark background (recurring).

## Summary of Defects

1.  **Code Block Background**: Mismatch with page theme (Global issue).
2.  **Sidebar Color**: Non-system blue (Global issue).
